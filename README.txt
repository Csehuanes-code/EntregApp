La aplicación EntregApp es un sistema desarrollado en Java que simula rutas de entrega en una ciudad utilizando estructuras de datos tipo grafo dirigido ponderado. Su propósito principal es modelar y optimizar las rutas de reparto para una red urbana, donde cada punto de entrega (nodos) y las conexiones entre ellos (aristas) representan intersecciones, almacenes, residencias o establecimientos comerciales, y los caminos o calles que los conectan.

Objetivos Principales:

Representar geográficamente una ciudad como un grafo.

Determinar rutas más cortas o eficientes desde un punto origen a múltiples destinos.

Evaluar diferentes escenarios de entrega según peso, tráfico, distancia o tiempo estimado.

Visualizar gráficamente los caminos recorridos por el repartidor o flota de reparto.

Características Técnicas:

Implementación de estructuras de datos como listas de adyacencia y matrices de costos.

Algoritmos de búsqueda y optimización de rutas como Dijkstra

Interfaz gráfica (GUI) intuitiva con Java Swing para mostrar el mapa de la ciudad, rutas calculadas y puntos de entrega.

Registro de eventos y estadísticas de entrega para análisis posterior.

Posibilidad de simular eventos como cierres de vías, demoras por tráfico o cambios de prioridad en entregas.

Aplicaciones prácticas:

Empresas de logística para planificar entregas urbanas.

Proyectos de investigación en optimización de rutas.

Simulación educativa para estudiantes de Estructuras de Datos, Teoría de Grafos y Optimización de Rutas.

Tecnologías utilizadas:

Lenguaje: Java (JDK 21+)

Estructuras: Grafos dirigidos ponderados

UI: Swing

Arquitectura: Orientada a objetos (POO)
